## 🚀 **Workflow VFX Simple**

### **1. Setup Projet**
```
Copier SQXX/ → SQ##/
Créer plans UNDLM_00XXX/ → UNDLM_00###/
```

### **2. Production**
```
Assets → Layout → Animation → Render
```

### **3. Livraison**
```
RENDERS/SQXX/FINAL/ → Export
```
